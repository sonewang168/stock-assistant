# 📱 股海秘書 v3.0

> 🇹🇼 台股 + 🇺🇸 美股 全功能 LINE Bot 投資助理

## ✨ 功能特色

### 📊 股票查詢
| 指令 | 說明 |
|------|------|
| `2330` | 查詢台股 |
| `AAPL` | 查詢美股 |
| `美 TSLA` | 查詢美股 |
| `查 台積電` | 搜尋股票 |

### 📈 大盤指數
| 指令 | 說明 |
|------|------|
| `大盤` | 台股加權指數 |
| `美股` | 美股三大指數 |
| `熱門` | 熱門台股 |
| `熱門美股` | 熱門美股 |
| `ADR` | 台積電 ADR 價差 |

### 📋 監控管理
| 指令 | 說明 |
|------|------|
| `+2330` | 加入監控 |
| `-2330` | 移除監控 |
| `監控` | 查看監控清單 |
| `監控 2330 1000 900` | 設定目標價 |
| `漲跌 2330 5` | 設定 ±5% 提醒 |

### 📊 技術分析
| 指令 | 說明 |
|------|------|
| `K 2330` | K 線走勢圖 |
| `技術 2330` | RSI + KD 指標圖 |

### 🤖 AI 功能
| 指令 | 說明 |
|------|------|
| `問 台積電前景如何？` | AI 對話 |
| `選股 高殖利率` | AI 選股 |
| `新聞 2330` | 股票新聞 |
| `財報 2330` | 財報摘要 |
| `PTT 2330` | 討論熱度 |

### 💼 投資組合
| 指令 | 說明 |
|------|------|
| `持股` | 查看持股 |
| `語音 2330` | 語音播報 |
